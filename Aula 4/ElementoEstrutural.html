Elementos estruturais - header

O header, também conhecido como cabeçalho, é uma parte essencial de um documento HTML. Ele é responsável por fornecer informações sobre o documento, como seu título, autor, data de criação, entre outros dados relevantes. Além disso, o header também pode conter elementos de navegação, como menus e links para outras páginas.
